# おかき鯖 メンバー紹介ページ (okaki-server member_list)

Minecraft風のデザインを採用した、おかき鯖メンバーの紹介ページです。  
GitHub Pages にて自動公開されています。

## 🛠 主な機能

- **Minecraftテーマ**: DotGothic16フォントとブロック調のデザイン。
- **自動ソート**: YouTubeのチャンネル登録者数が多い順（降順）に自動整列。
- **ヘッダーロゴ演出**: モバイル版では一定量スクロールするとロゴが出現。
- **自動更新 (GitHub Actions)**:
    - **週次 (月曜 0:00)**: YouTube Data API を使用して、おすすめ動画（チャンネル予告編）とアイコン画像を自動更新。
    - **日次 (毎日 0:00)**: 全メンバーの最新登録者数を取得し、表示順に反映。

## 👥 メンバーの追加・編集方法

メンバーの更新は `members.json` を編集することで行います。

### 1. `members.json` を編集する
以下のように、配列の末尾に新しいオブジェクトを追加してください。

```json
{
  "id": "new_member_id",
  "name": "メンバー名",
  "youtube": "https://www.youtube.com/@ChannelHandle",
  "x": "https://x.com/TwitterID",
  "generation": 1
}
```

- **必須項目**: `id`, `name`, `youtube`, `x`
- **自動取得される項目** (入力不要):
    - `icon`: YouTubeのアイコンを自動ダウンロードし、相対パわで保存されます。
    - `subscribers`: 最新の登録者数が毎日更新されます。
    - `rec`: チャンネルの「おすすめ動画ID」が週次で自動取得されます。
    - `recTitle`: おすすめ動画のタイトルが週次で自動取得されます。

### 2. 変更を Push する
GitHub 上で編集してコミットするか、ローカルから Push すると、次回の自動実行時に残りの情報が補完され、サイトが更新されます。  
※ すぐに反映させたい場合は、GitHub の Actions タブから `Weekly Maintenance` を手動実行してください。

## ⚠️ 注意事項

- **文字コード**: `members.json` は必ず **UTF-8** で保存してください。
- **APIの上限**: YouTube API のクォータ制限があるため、大量のメンバーを一気に追加した直後は取得に時間がかかる場合があります。
