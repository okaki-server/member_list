# おかき鯖 メンバー紹介サイト

kaki・たけぉが運営する『おかき鯖』のメンバー紹介用Webサイトです。

## 主な機能
- **動的メンバーリスト**: `members.json` から情報を読み込み、登録者数順に並べて表示します。
- **インベントリ風UI**: マインクラフトをイメージしたデザインのメンバーカード。
- **詳細モーダル**: 各メンバーのアイコン、登録者数、SNSリンク（YouTube、X）、おすすめ動画を表示します。
- **サブチャンネル対応**: 1人のメンバーに対して複数のYouTubeチャンネルを表示可能です。
- **自動データ更新**: GitHub Actions を利用して、登録者数を定期的に自動更新します。
- **OGP対応**: SNS共有時にサイトのプレビューを表示するためのメタタグと専用画像を完備。

## 開発・メンテナンス

### メンバー情報の更新
`members.json` を編集します。

### アイコンの更新
新しいメンバーを追加した場合は、以下の手順でアイコンを更新・保存します：
1. `node scripts/fetch_icons.js` を実行してYouTubeから最新のアイコンURLを取得します。
2. `node scripts/download_icons.js` を実行して、取得したURL先の画像を `icons/` ディレクトリにローカル保存します。

### 登録者数の更新
`node scripts/update_subscribers.js` を実行します（YouTube Data APIキーが必要）。
- **複数チャンネル対応**: `youtube` フィールドが配列の場合、すべてのチャンネルから登録者数を取得し、最も多い方の数値を採用します。また、該当のチャンネルを配列の先頭（メイン）に入れ替えます。
- ローカルでは `.env` ファイルに `YOUTUBE_API_KEY` を設定してください。

### オススメ動画の更新
`node scripts/update_recommendations_api.js` を実行します。
- **手動設定の維持**: `members.json` 内で `"isRecManual": true` が設定されているメンバーは、自動更新の対象から外れ、現在の `rec` / `recTitle` が維持されます。
